# numLuau
numluau (numerical luau) is a math library for luau.

features:
- a N dimensional array object
    - broadcasting
    - sophisicated slice indexing
    - element wise operations

- linear algebra
    - linear equation solvers
    - eigen value/vectors
    - QR and LU decompositions
    - determinants
    - dot/cross products
    - matrix multiplications

and alot more
